# Idea Video Pendek AI (BM)

Jana idea video pendek AI (TikTok/Reels/Shorts) yang padu dalam Bahasa Melayu — lengkap dengan hook, struktur, visual, audio, CTA, hashtag dan caption.

## Cara Jalankan

Ini ialah laman statik tanpa dependensi.

- Buka `index.html` dalam pelayar; atau
- Guna Vercel CLI:
  ```bash
  vercel build
  vercel deploy --prod --yes --name agentic-fb9c4acf
  ```

## Struktur

```
├── index.html
├── styles.css
├── script.js
└── vercel.json
```

## Ciri

- Generator 10 idea dan skrip penuh 30–60s
- Pilihan platform, durasi, tona, niche dan trend
- Butang Salin Semua dan Eksport .txt
- Reka bentuk responsif dan pantas
